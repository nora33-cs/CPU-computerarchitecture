# Custom CPU Design & Simulation

## Overview

This project involves the design and simulation of a custom Central Processing Unit (CPU) as part of a Computer Architecture course. The CPU was developed to execute a set of instructions by integrating key processor components such as the Arithmetic Logic Unit (ALU), registers, control unit, and memory interfaces.

## Features

* Custom instruction set architecture (ISA)
* Arithmetic and logical operations
* Register-based data storage and manipulation
* Control unit for instruction decoding and execution
* Support for memory access instructions
* Simulation and verification of processor functionality

## System Components

* Arithmetic Logic Unit (ALU)
* Register File
* Program Counter (PC)
* Control Unit
* Instruction Memory
* Data Memory
* Multiplexers and Datapath Components

## Technologies Used

* Digital Logic Design
* Computer Architecture Principles
* Logic Simulation Tools (add the tool you used)

## Project Objectives

* Understand CPU organization and datapath design
* Implement instruction execution cycles
* Analyze processor behavior and performance
* Apply computer architecture concepts in a practical system

## Results

Successfully designed and tested a functional CPU capable of executing arithmetic, logical, memory, and control-flow instructions through simulation.
